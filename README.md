## 概念

`Document` 的概念, 单个Document 可以了理解成 一个主体，在这个主体中包含了很多的记录，而我们可以通过这个
Document 来进行增删改查，这个主体可以是一个用户，一个订单，一个商品等等。

那么对于前端来说，日常开发中，我们会有很多的跨组件或者跨页面的数据需要共享，这个时候我们就需要一个全局的
Document 来进行管理，而这个全局的 Document 就是我们的 `Store`。
